ImportError: No module named pytesser


没有安装pytesser
百度上我也找了很多教程，写这篇文章就是要说说安装pytesser遇到的坑。。。

安装Pytesser

可以在Pytesser的官网下载，也可以在这里百度云盘下载。
将压缩包文件解压到C:\Python27\Lib\site-packages\下，文件名去掉后面的版本信息，改为pytesser。
打开pytesser文件夹，将pytesser.py修改成__init__.py(注意左右都是两个下划线，我就是因为看成了一个搞了半天)，打开py文件进行修改:

import Image   改为
from PIL import Image
tesseract_exe_name = 'tesseract'  改为 
tesseract_exe_name = 'Python安装路径\\Lib\\site-packges\\pytesser\\tesseract
